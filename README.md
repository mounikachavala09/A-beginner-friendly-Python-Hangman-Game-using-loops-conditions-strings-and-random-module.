# A-beginner-friendly-Python-Hangman-Game-using-loops-conditions-strings-and-random-module.
Hangman Game is a simple text-based Python project where the player guesses a hidden word letter by letter within limited attempts. It uses loops, conditions, strings, lists, and the random module to create an interactive console game and improve basic Python programming skills.
Hangman Game in Python
A simple text-based Hangman Game developed using Python.
In this game, the player tries to guess a hidden word one letter at a time within a limited number of attempts.
Features :
1.Random word selection
2.Letter-by-letter guessing
3.Limited wrong attempts
4.Input validation
5.Simple console-based interface

Technologies Used:
1.Python
2.Random module
3.Loops and Conditional Statements

Concepts Covered:
1.Strings
2.Lists
3.While loops
4.If-Else conditions
5.User Input Handling
How to Run:
python hangman.py
Learning Outcome
This project helped in understanding basic Python programming concepts and improving logical thinking skills through game development.
